## About making a Webpage:

*   <a href="https://neocities.org" target="_blank">Neocities</a>: This is where I host this page, explore the learning tab to know different SSG, CSS frameworks and more!!! Or just click on the webpages that You like.
*   <a href="https://www.theodinproject.com/" target="_blank">The Odin Project</a>: If You like how to make a webpage from the ground up... I don't think is a better resource than that but be aware!!! It's hard and You will need a good amout of time to complete just the Fundations path.
*   <a href="https://www.11ty.dev/" target="_blank">11ty</a>: It's the Static Site Generator that I use.
*   <a href="https://www.11ty.dev/" target="_blank">11ty Tutorials</a>: Check out <a href="https://flamedfury.com/guides/11ty-homepage-neocities/" target="_blank">Flammed &quot;Create A Static Site...&quot;</a> and <a href="https://afellowu.neocities.org/blog/11ty-github-and-neocities" target="_blank">Afellowu &quot;11ty, GitHub and Neocities&quot;</a>—They are two awesome guides on how to get started.
*   <a href="https://pika.page/" target="_blank">Pika</a>: If you don't want to build a blog with an SSG, this is another great option. For the time being, I prefer keeping everything on my page. 
*   <a href="https://caffeineandlasers.com/blogs/automatingYourRSS.html" target="_blank">RSS Automatizaton</a>: A very simple way to automate your RSS feed and display links to your blog posts in a list anywhere.
*   <a href="https://strflr.neocities.org/blog/posts/2025-02-25-Image-Optimization/" target="_blank">Image optimization</a>: If you work with large artwork, this is a lifesaver (also, check out their site in general—the art is great).
*   <a href="https://www.11ty.dev/docs/quicktips/tag-pages/" target="_blank">Zero Maintenance Tag Pages for your Blog</a> and <a href="https://joesahlsa.dev/blog/eleventy-tag-filter/" target="_blank">Eleventy tag filter</a>: Perfect for a simple and organized tag system.
