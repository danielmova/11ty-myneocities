## Acerca de la creación de páginas web:

*   <a href="https://neocities.org" target="_blank">Neocities</a>: Aquí es donde está alojada esta página, la sección learning tiene listados de herramientas imprescindibles para crear una página web y un tutorial bastante básico pero funcional.
*   <a href="https://www.theodinproject.com/" target="_blank">The Odin Project</a>: Si lo que te interesa es aprender desde cero cómo funciona una página web... No hay mejor lugar que ese curso, si es largo, si está inglés pero te lleva de cero a manejar Linux, aprender HTML, CSS y Javascript de la mejor manera posible.
*   <a href="https://www.11ty.dev/" target="_blank">11ty</a>: Es el Static Site Generator que uso... Si no sabes que es un Static Site Generator... Es magia negra para crear páginas web sin perder la cabeza en el proceso.
*   <a href="https://www.11ty.dev/" target="_blank">Tutoriales acerca de 11ty</a>: <a href="https://afellowu.neocities.org/blog/11ty-github-and-neocities" target="_blank">Afellowu &quot;11ty, GitHub and Neocities&quot;</a> Este es el tutorial más completo, te lleva desde cero a tener automatizado la actualización de tu página web. <a href="https://flamedfury.com/guides/11ty-homepage-neocities/" target="_blank">Flammed &quot;Create A Static Site...&quot;</a> No puedo dejar atrás el tutorial de Flammed, está desactualizado pero es claro como el cristal... Los dos están en inglés... La mayoría de cosas interesantes están en inglés... 
*   <a href="https://pika.page/" target="_blank">Pika</a>: Si prefieres no tener que preocuparte por mantener una página web para un blog sencillo, Pika funciona muy bien. tiene un límite pero primero escribe 50 blogs y me dices si esos límites realmente importan... 
*   <a href="https://caffeineandlasers.com/blogs/automatingYourRSS.html" target="_blank">RSS Automatizaton</a>: Un tutorial sencillo y al punto para tener RSS automatizado en 11ty. 
*   <a href="https://strflr.neocities.org/blog/posts/2025-02-25-Image-Optimization/" target="_blank">Image optimization</a>: Esto es para mi porque se que en algún punto voy a tener una imagen gigante que voy a tener que optimizar de alguna forma.
*   <a href="https://www.11ty.dev/docs/quicktips/tag-pages/" target="_blank">Zero Maintenance Tag Pages for your Blog</a> y <a href="https://joesahlsa.dev/blog/eleventy-tag-filter/" target="_blank">Eleventy tag filter</a>: Un sistema de etiquetas "sencillo"... me dió muchos dolores de cabeza pero funciona.
